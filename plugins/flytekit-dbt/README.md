# Flytekit dbt plugin

Flytekit plugin for performing DBT tasks. Currently it supports  `dbt run` , `dbt test`, `dbt source freshness` tasks.

To install the plugin, run the following command:

```bash
pip install flytekitplugins-dbt
```
