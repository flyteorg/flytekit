Implement Workflows
********************
Workflows are purely declarative entities that construct a DAG of tasks using the data flow between the tasks.

- Versioned (Usually tied to the git sha)
- optional strong interfaces (specified inputs and outputs)
- declarative
- executable
- unit testable
- can be scheduled using LaunchPlans
